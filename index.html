JSONBin Setup
Not connected — tasks stored locally only

JSONBin is a free cloud service for storing JSON data. Your tasks sync across any device in seconds. Takes about 2 minutes to set up.

1
Create a free JSONBin account
Go to jsonbin.io and sign up for free. No credit card needed.
2
Get your API key
After signing in, click your profile icon (top right) → API Keys.
Copy your Secret Key (starts with $2a$...).
3
Paste your API key below and click Connect
The app will create a new bin automatically and start syncing. That's it.
